# Ofuq Project - منصة أفق لريادة الأعمال

## نبذة عن المشروع
منصة أفق هي نظام تقني متكامل يهدف إلى دعم بيئة العمل الريادية، وتسهيل عملية إدارة المشاريع من خلال توفير ربط سحابي متطور. تم تصميم المنصة لتعمل كحلقة وصل بين المبدعين والبيانات السحابية الموثوقة.

## التقنيات المستخدمة
تم الاعتماد في بناء هذا المشروع على تقنيات حديثة لضمان الأداء العالي والأمان:
* **Frontend**: تم تطوير واجهة المستخدم باستخدام تقنيات الويب الأساسية لضمان سرعة التجاوب.
* **Backend & Database**: تم استخدام منصة **Supabase** كقاعدة بيانات سحابية متكاملة (Backend-as-a-Service)، حيث تم تصميم جداول البيانات (Tables) وإدارة المصادقة (Authentication) لضمان أمن البيانات.
* **Security**: تم تفعيل بروتوكولات حماية الصفوف (Row Level Security - RLS) لضمان حماية خصوصية بيانات المستخدمين.

## المميزات التقنية
1. **الربط السحابي (Cloud Integration)**: ربط آمن وسريع مع Supabase عبر مفاتيح API معتمدة.
2. **إدارة البيانات**: هيكلة قواعد بيانات مرنة تسمح بإضافة وتعديل المشاريع (Projects) بشكل لحظي.
3. **المزامنة الحية**: يدعم النظام التزامن مع العمليات التي تتم في قاعدة البيانات لضمان دقة البيانات المعروضة.

## خطوات التشغيل والتثبيت
للحصول على نسخة محلية من المشروع:
1. قم بنسخ المستودع (Clone the repository).
2. تأكد من ضبط إعدادات الاتصال الخاصة بـ Supabase (URL & API Key) في ملفات الإعداد الخاصة بالمشروع.
3. قم بإنشاء الجداول اللازمة في قاعدة البيانات وفق المخطط المعتمد في المشروع.

## المسؤولية
تم تنفيذ هذا المشروع كجزء من المتطلبات الأكاديمية لتحقيق الكفاءة في استخدام أدوات التطوير السحابي وإدارة قواعد البيانات.
